# Linux tarixi

:::tip Eslatma:
Eslatma: ushbu mavzu [linuxjourney](https://linuxjourney.com/lesson/linux-history) saytidan tarjima qilindi.
:::


Salooooom yangilar!, rostdan ham Linux deb nomlanuvchi ushbu ajoyib dunyoga kirishga qaror qildingizmi ? Oldinda bizni ajoyib va qiziqarli sayohat kutmoqda! Mening ismim [Pingvin Pit](https://linuxjourney.com/assets/jumbotron-60c93c2fc6bb16079b70fe2eccacaa05f2d6b002af4ff0667ddd8e283cbf81fd.png) va men sizga ushbu sayohatda boshchilik qilish uchun shu yerdaman. Keling, Linux haqida qisqacha ma'lumot bilan boshlaylik.


Linux qanday paydo bo'lganligi haqida bilish uchun, Bell Laboratories xodimi [Ken Tompson](https://i.imgur.com/ZXdBFXM.png) va [Dennis Richi](https://i.imgur.com/NsnEUCr.png) UNIX operatsion tizimini ishlab chiqqan 1969 yil boshiga qaytaylik.  Linux keyinchalik portativ bo'lishi uchun C dasturlash tilida qayta yozildi va keng tarqalgan operatsion tizimga aylandi.

Oradan o'n yil o'tgach, `Richard Stallman` [GNU (GNU is Not UNIX)](https://www.texnoman.uz/post/gnu-linux-distributivlari-haqida.html) loyihasi ustida ishlay boshladi, GNU yadrosi Hurd deb nomlangan,lekin afsuski, u tugallanmadi. Buning natijasida bepul dasturiy ta'minot litsenziyasi bo'lgan GNU General Public License (GPL) yaratildi.

Yadro - operatsion tizimning eng asosiy qismi hisoblanadi. U qurilmaning dasturiy ta'minot bilan aloqa qilishiga imkon beradi. Bundan tashqari, u juda ko'p boshqa amallarni bajara oladi, lekin biz bu haqida boshqa mavzuda o'rganamiz. Hozircha, yadro sizning tizimingizda sodir bo'layotgan deyarli hamma narsani boshqarishini bilishingiz kerak.

Bu vaqt ichida BSD, MINIX kabi UNIX-ga o'xshash va boshqa tizimlar ishlab chiqildi. Biroq, UNIX-ga o'xshash tizimlarning barchasini umumiy birlashtiruvchi yadro yo'q edi.

Keyin 1991 yilda [Linus Torvalds](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTPEIkzbmUdllgB-RBNfpo5aFsFXJ1uTUfoqByYovgh9w&s) ismli yosh yigit bugungi kunda biz bilgan Linux yadrosini ishlab chiqa boshladi.
